# Pong 1.0

![Pong1](https://github.com/TsuHub/Pong-Game-1/blob/master/Pong1.jpg?raw=true)

<p align="left">
	1 - no terminal, dar o comando

	cd <diretório src>


	2 - já no diretório, dar o comando

	javac Ball.java
	javac Player.java
	javac Score.java

	(ou: $javac *.java)


	3 - no diretório, para a execução do programa,
	dar o comando:

	java Pong

</p>
